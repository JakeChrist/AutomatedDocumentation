How to Use It

This module implements a command-line interface for generating documentation using DocGen-LM. It scans a source tree for Python and MATLAB files, parses them, requests summaries from a running LLM, and writes HTML documentation. The module includes functions for cleaning the output directory, summarizing text, tokenizing, chunking text, summarizing modules, rewriting docstrings, and recursively summarizing classes and methods.

System Requirements

This Python module provides a parser for Python files used by DocGen-LM. It uses the `ast` module to extract structured information from Python code according to the Software Requirements Specification (SRS). The module includes functions to format function signatures, parse classes and functions, and parse entire Python source files into structured dictionaries.
